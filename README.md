# Portfolio Website

A personal portfolio website showcasing my skills, education, experience, achievements, and contact information. The website is built using HTML, CSS, and JavaScript, featuring a dark mode for enhanced user experience.

## Sections

- **About:** Brief introduction about myself.
- **Education:** Details about my educational background.
- **Experience:** Highlights of my professional experience.
- **Achievements:** Notable accomplishments and recognitions.
- **Skills:** List of skills and competencies.
- **Contact:** Ways to get in touch with me.

## Dark Mode

The portfolio website includes a dark mode for a visually pleasing experience, especially in low-light environments.

## Contact Form

The contact section features a form powered by Web3Forms, making it easy for visitors to reach out.

## Installation

1. Clone the repository.
2. Open the `index.html` file in your preferred web browser.

## Usage

Explore the different sections of the portfolio to learn more about my background and skills. Feel free to use the contact form for inquiries or collaboration.
